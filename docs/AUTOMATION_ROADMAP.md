# Automation roadmap

## Stage 1 — included now

- One episode JSON as the source of truth.
- PowerShell orchestration.
- Static website generation.
- Social image generation.
- Pinterest CSV generation.
- Git-based deployment.

## Stage 2 — low-risk automation

- Pull YouTube title, thumbnail and duration from a local metadata file.
- Generate scheduled social calendars.
- Add an episode publication log.
- Add image templates for multiple visual treatments.

## Stage 3 — authenticated APIs

- Instagram Graph API publishing.
- Pinterest API publishing after production access.
- Search Console sitemap submission.
- Optional Blogger syndication.

Store API credentials in environment variables or GitHub Secrets. Never commit tokens to the repository.
